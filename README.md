# ネイルサロンMARINE ランディングページ

## プロジェクト概要
千葉県のメンズ専門ネイルサロン「MARINE」のランディングページ作成

## ミッション
既存のindex.htmlを改良し、20-30代男性をターゲットにしたLINE登録促進のLPを作成

## 作業指示
1. 既存のindex.htmlを分析
2. requirements.mdの要件に基づいて改良
3. content.jsonのデータを使用してコンテンツを充実
4. design-guide.mdに従ってスタイリング
5. レスポンシブ対応必須

## 完成時の確認ポイント
- LINE登録ボタンが複数箇所に配置されている
- メンズ専門であることが明確
- 料金が分かりやすく表示されている
- スマホでの表示が最適化されている
